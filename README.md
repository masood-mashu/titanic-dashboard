# 🚢 Titanic Survival Dashboard

An interactive Data Science dashboard analyzing survival patterns 
of Titanic passengers, built with Python and Streamlit.

## 🔗 Live Demo
👉 [Click here to view the live app](https://titanic-dashboard-analysis.streamlit.app)

## 📊 Project Overview
This project performs Exploratory Data Analysis (EDA) on the 
famous Titanic dataset from Kaggle. The dashboard allows users 
to interactively filter and explore survival patterns based on 
gender, passenger class, and age.

## 🔍 Key Insights
- Only **38.4%** of passengers survived
- **Women** had a much higher survival rate than men
- **Class 1** (rich) passengers survived more than Class 3 (poor)
- Younger children had better survival rates

## 🛠️ Technologies Used
- Python
- Pandas — data manipulation and cleaning
- Matplotlib & Seaborn — data visualization
- Streamlit — interactive web dashboard
- GitHub — version control
- Streamlit Cloud — deployment

## 📁 Dataset
- Source: [Kaggle - Titanic Dataset](https://www.kaggle.com/competitions/titanic)
- 891 passengers, 12 features

## 🚀 How to Run Locally
1. Clone the repository
   git clone https://github.com/masood-mashu/titanic-dashboard.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run dashboard.py
